專案作品
用具體專案展示後端與資料庫能力
會員管理系統（練習專案）
角色：個人開發 狀態：持續優化中
Java MySQL CRUD

專案內容： 練習設計一個簡單的會員管理系統，包含新增、查詢、更新、刪除會員資料等功能， 藉由實作熟悉 CRUD 流程與資料表設計。

負責內容： 使用 Java 撰寫基本後端邏輯，搭配 MySQL 建立資料表結構與關聯，並撰寫 SQL 來完成資料操作。

學到的重點： 更理解資料表欄位與索引設計的重要性，以及在程式邏輯中如何處理例外狀況與錯誤訊息。

書籍管理 REST API 與 Redis 快取優化系統
角色：個人開發 類型：學習型專案
Spring Boot Spring Data JPA Redis MySQL Docker Compose

專案內容： 使用 Spring Boot、Spring Data JPA 與 Redis 建立書籍管理 REST API， 實作新增、查詢、修改、刪除等 CRUD 功能。

快取機制設計： 針對單筆查詢導入 Redis Cache-Aside 策略，查詢時優先讀取 Redis， 未命中時從 MySQL 讀取並回寫快取，以降低資料庫壓力、提升查詢效能。

資料一致性處理： 在更新與刪除資料時同步清除對應快取，避免讀到過期資料，確保 MySQL 與 Redis 資料一致。

資料表管理： 支援 TRUNCATE TABLE 全表清空與 ID 重置，並同步清除相關快取資料。

環境部署： 使用 Docker Compose 整合 Spring Boot、MySQL 與 Redis，建立可快速部署的開發環境。

學習重點： 實作過程中強化了對 RESTful API 設計、快取策略（Cache-Aside）、資料一致性與容器化部署的理解， 也更熟悉如何在真實專案中平衡效能與維護成本。

Linux 環境練習（部署與指令）
角色：個人練習 主題：系統操作
Linux Shell Server

專案內容： 在 Linux 環境中練習基本指令操作、檔案權限設定，以及簡單的服務啟動與管理， 為日後部署後端服務做準備。

負責內容： 建立使用者、設定檔案權限、安裝必要套件，並撰寫簡單腳本協助操作。

學到的重點： 更熟悉命令列操作與系統結構，理解檔案權限與服務管理對系統穩定性的影響。
